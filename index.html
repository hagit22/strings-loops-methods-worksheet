פעולות עם פרמטרים וערכי החזרה
מה עושים בשלב הזה? לוקחים את הקוד שכתבתם ב- LoopsAndStrings.cs וכותבים אותו מחדש בצורה אחרת.

הפעם: ה- Main אחראי על קליטה מהמשתמש ועל הדפסת התוצאה.
הפעולות אחראיות רק על החישוב – מקבלות נתונים כפרמטרים ומחזירות תוצאה.

קובץ חדש: צרו פרויקט חדש ב-Visual Studio. שם הקובץ: LoopsStringsAndMethods.cs
תרגיל 1 – ספירת מופעים של תו
הפעולה תקבל מחרוזת ותו, ותחזיר כמה פעמים התו מופיע במחרוזת. ה- Main יקלוט את הנתונים, יקרא לפעולה, ויקבל ממנה את התוצאה ויציג אותה.
שלב א׳ – מה ה- Main צריך לעשות?

חשבו: אם הפעולה לא קולטת כלום מהמשתמש – מי קולט? מה בדיוק ה- Main צריך לקלוט?

ה- Main קולט את שני הנתונים: המחרוזת והתו. לאחר מכן הוא יעביר אותם לפעולה.
Console.Write("Enter a string: "); string text = Console.ReadLine(); Console.Write("Enter a character: "); char tav = Console.ReadLine()[0];
שלב ב׳ – מה הפעולה מקבלת כפרמטרים?

הפעולה צריכה לחשב את מספר המופעים של התו. אילו נתונים היא צריכה לקבל כדי לעשות את זה?

היא צריכה לקבל את המחרוזת ואת התו – אלו הנתונים שהיא צריכה לחישוב.
החתימה תראה כך: static ??? CountOccurrences(string text, char tav)
שימו לב לסימני השאלה – מה צריך להיות שם?
שלב ג׳ – מה הפעולה מחזירה?

הפעולה מחשבת כמה פעמים התו מופיע. מה היא צריכה להחזיר? ומה הטיפוס של הערך הזה?

היא מחזירה מספר שלם – מספר המופעים. לכן הטיפוס הוא int.
החתימה המלאה של הפעולה: static int CountOccurrences(string text, char tav)
שלב ד׳ – איך הפעולה מעבירה את התוצאה חזרה ל- Main?

הגדרנו שהפעולה מחזירה int – אבל איפה בקוד זה קורה בפועל?

בסוף הפעולה, אחרי הלולאה, כותבים: return count;
זו הפקודה שמעבירה את הערך בחזרה למי שקרא לפעולה.
שלב ה׳ – מה משתנה בגוף הפעולה?

גוף הפעולה כמעט זהה לקוד הישן – אבל יש דבר אחד שצריך להסיר. מה זה?

מסירים את קליטת הנתונים מהמשתמש (Console.ReadLine) ואת ההדפסה (Console.WriteLine).
הפעולה רק עושה את הלולאה והספירה, ומחזירה את count.
שלב ו׳ – איך ה- Main מקבל את התוצאה ומדפיס אותה?

ה- Main קורא לפעולה ומעביר לה את הנתונים. הפעולה מחזירה ערך – מה עושים איתו?

שומרים את הערך המוחזר במשתנה, ואז מדפיסים:
int result = CountOccurrences(text, tav); Console.WriteLine("Character appears " + result + " times");

מה ה- Main קולט? מה הפעולה מקבלת כפרמטר?
ה- Main קולט את המשפט. הפעולה מקבלת פרמטר אחד: string sentence.
מה הפעולה מחזירה?
מספר שלם – מספר המילים. חתימת הפעולה: static int CountWords(string sentence)

הפעולות הקודמות החזירו int. מה מחזירה הפעולה הזו?

הפעם הפעולה מחזירה מחרוזת – את הטקסט עד הנקודה. לכן הטיפוס הוא string.

איך בונים את מחרוזת התוצאה?

תרגיל 5 – פלינדרום
הפעולה תקבל טקסט ותחזיר האם הוא פלינדרום. ה- Main יקלוט, יקרא לפעולה, ויציג.

הפעולה מחזירה true או false – כלומר טיפוס bool.

בגוף: אם מצאתם זוג שלא תואם – return false;. בסוף הלולאה: return true;
